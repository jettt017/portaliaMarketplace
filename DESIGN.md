# DESIGN.md — PORTALIA

## Overview

Portalia uses a modern mobile marketplace design approach with clean UI, soft gradients, premium spacing, and user-focused experiences. The interface is designed to feel modern, lightweight, and intuitive for university students.

---

# Design Principles

## 1. Clean & Minimal

The interface should remain simple, uncluttered, and focused on the main content.

## 2. Clear Visual Hierarchy

Every screen must have a clear visual order:

* Header
* Primary Content
* CTA (Call To Action)

## 3. Soft & Friendly

Use rounded corners, soft shadows, and pastel colors to create a comfortable and modern experience.

## 4. Touch Friendly

All buttons and inputs should be easy to interact with on mobile devices.
